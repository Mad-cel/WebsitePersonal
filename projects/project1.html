AI Article Classifier

Detecting AI-generated articles using machine learning models.

Project Overview

This project uses Random Forest, KNN, and Logistic Regression to analyze text and determine whether an article was generated by an AI.

Technologies Used

  • Python (Scikit-Learn, NLTK)
  • Machine Learning Models: Random Forest, KNN, Logistic Regression
  • Flask API for deployment
